# OMN Exp #
===========

To initialize your local repository, use this command:

	repo init -u https://github.com/omnexp/android_manifest.git -b 8.1

Build omnexp

	. build/envsetup.sh
        lunch omnexp_$device-userdebug
        mka omnexp -jx (x = number cpus)
